API信息的查询与配置

对于需要使用领星开放接口的企业，可以根据此文档获取自己企业的开发者账号，进行基础信息的查询与配置

1. 如何申请企业的AppID和AppSecret？

使用超级管理员账号登录领星ERP，进入【设置】>【业务配置】>【全局】>【开放接口】，输入邮箱进入开放接口后台即可自动生成； 

2. 如何查看企业的AppID和AppSecret？

使用超级管理员账号登录领星ERP，进入【设置】>【业务配置】>【全局】>【开放接口】，输入邮箱用于接收领星开放平台相关通知消息，点击【进入开放后台】按钮

成功开通的客户，可直接在【API信息】中查看企业的AppID和AppSecret

请注意：获取该AppID和AppSecret将有权限访问企业全部数据，请妥善处理，避免外泄！如需作废可点击 重置 按钮作废并生成新的授权码

3. 如何配置IP白名单？

使用超级管理员账号登录领星ERP，进入【设置】>【业务配置】>【全局】>【开放接口】页配置

请注意：在配置IP白名单时，请添加您的公网IP地址配置的白名单是真实有效的。通过此链接快速查询您的公网IP：https://toolbox.lingxing.com/api/getIp

4. 其他说明

IP白名单是否支持域名？
IP白名单不支持域名形式

因人员离职或其它原因，需要作废原有的AppSecret怎么办？
可以使用超级管理员账号登录领星ERP，在【设置】>【业务配置】>【全局】>【开放接口】中，点击【重置】AppSecret，生成新的AppSecret后，原AppSecret将立即失效
