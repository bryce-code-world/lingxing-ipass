按ASIN查询FBA补货建议图表

支持按ASIN查询系统中FBA补货建议，获取未来销量预测和库存预测

接口信息
API Path	请求协议	请求方式	令牌桶容量
/erp/sc/routing/fbaSug/asin/getDailySalesInfoFeature	HTTPS	POST	1
请求参数
参数名	说明	必填	类型	示例
sid	店铺id ，对应查询亚马逊店铺列表接口对应字段【sid】	是	[int]	136
asin	ASIN	是	[string]	B0xxxxxxxx
sug_type	建议类型：
1 建议采购量
2 建议本地仓发货量
3 建议海外仓发货量	是	[int]	3
mode	补货建议模式：
0=普通模式
1=海外仓中转模式
不传默认取系统当前设置模式	否	[int]	1
请求示例
{
    "sid": 136,
    "asin": "B0xxxxxxxx",
    "sug_type": "3",
    "mode": "1"
}
复制
错误
复制成功
返回结果

Json Object

code	状态码，0 成功	是	[int]	0
message	消息提示	是	[string]	success
error_details	错误信息	是	[string]	
request_id	请求链路id	是	[string]	C3D9F541-8083-E376-EB5C-606A872F5C89
response_time	响应时间	是	[string]	2022-12-08 18:27:13
total	总数	是	[int]	1
data	响应数据	是	[object]	
data>>list	每日数据信息【按天返回360天的数据】	是	[object]	
data>>list>>日期	每日数据，[到货量,当日销量,当时实时库存]	是	[array]	[2194,0,"2194.00"]
data>>sug_date_line	图表时间线	是	[array]	2021-07-26
data>>sug_date_line>>date	日期	是	[string]	
data>>sug_date_line>>desc	上个日期到当前区间描述	是	[string]	xxxx
data>>sug_date_line>>title	标题	是	[string]	xxx
data>>sug_date_line>>type	日期类型：
oversea_arrive_date 海外仓到货日
with_safe_day_date 安全天数日
with_days_frequency_date 补货频率日
